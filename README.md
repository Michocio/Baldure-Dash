<h1> Remake of Baldure Dash </h1>

Format of board items:
* EMPTY FIELD ' '
* SAND '.'
* WALL '#'
* STONE 'O'
* DIAMOND '$'
* EXIT 'X'
* GAMER '&'

As in orginal game, stone are falling down if there is no ground under (that's gravity, you know ;) ).
